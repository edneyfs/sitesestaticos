Nomenclatura do Arquivo

Cada arquivo deve seguir o padrão: <layout>_<timestamp>.json

Onde <layout> é o nome do layout (plural) e <timestamp> segue o formato yyyyMMddHHmmssSSS (17 dígitos).

Formatos suportados: .json (array), .jsonl (uma linha por registro), .json.gz e .jsonl.gz (comprimidos).

Exemplos: categories_20260514114401000.json, products_20260514114403000.jsonl.gz

Estrutura de Pastas (S3)
bucket/
├── incoming/     ← Depositar arquivos aqui
├── processing/   ← Em processamento (gerenciado pelo sistema)
├── processed/    ← Processados com sucesso
├── error/        ← Processados com erro
└── reports/      ← Relatórios JSON de cada processamento
